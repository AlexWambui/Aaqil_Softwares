Introduction

What is JavaScript?

JavaScript is a programming language that adds interactivity to your website by giving it a dynamic touch instead of just being static.

This happens in games, in the behavior of responses when buttons are pressed or with data entry or form validation; with dynamic styling; with animations to create interactive content. etc.

It was invented by Brendan Eich (co-founder of the Mozilla project, the Mozilla Foundation and the Mozilla Corporation) in 1995.

With more experience you can: create games, animated 2D and 3D graphics, comprehensive database-driven apps etc.

Without any frameworks, plain JavaScript is called vanilla JavaScript.

A lot of tools have been written on the core JavaScript language, unlocking a vast amount of functionality with minimum effort including:

  • Browser Application Programming Interfaces (APIs) built into web browsers, providing functionality such as dynamically creating HTML and setting CSS styles; collecting and manipulating a video stream from a user's webcam, or generating 3D graphics and audio samples.
  • Third-party APIs that allow developers to incorporate functionality in sites from other content providers such as Twitter or Facebook.
  • Third-party frameworks and libraries that you can apply to HTML to accelerate the work of building sites and applications.

As your JS skills grow, your website will enter a new dimension of power and creativity.

Getting comfortable JS can be challenging so you have to start small and progress gradually.

JavaScript is a case sensitive language.

JavaScript is a scripting language for HTML and this enables us to reduce the load on the server as we can perform actions on the clinet machine (browser) without having to load the server with some operations. Take an example of adding two numbers, you can just do it on the client machine using JavaScript instead of having to load this numbers from the servers then performing the calculation.

Uses of JavaScript

  • Gaming
  • Mobile applications
  • Server-side scripting
  • Client-side scripting

A hello, world! example

Example of JS Code in the HTML Code:

JavaScript code must be inserted between the <script> tag when used in HTML files.

JavaScript code can be placed in both the head (external embedding) and body (internal embedding) elements.

                        
                            <!DOCTYPE html>
                                <head>
                                    <title>JavaScript</title>
                                </head>
                                <body>
                                    <script>
                                        document.write("Hello, World!");
                                    </script>
                                </body>
                            </html>

For the html page, add this code just before the </body> tag. This alerts the browser to start interpreting all the text between these tags as a script.

this does the same job as the <link> element for CSS. It applies the JS to the page, so it can have an effect on the HTML (along with the CSS, and anything else on the page).

                        
                            <script src="scripts/main.js"></script>

Create a separate JS file called main.js and add this code:

                        
                            const myHeading = document.querySelector('h1');
                            myHeading.textContent = 'Hello, World!';

Variables

Variable is an item of data that has a name and a value.

variables are containers that store values. Like a number we might use in a sum, or a string that we might use as part of a sentence.

Variables are necessary to do anything interesting in programming.

If values couldn't change, then you couldn't do anything dynamic, like personalizing greeting message or changing an image in an image gallery.

Variables may hold values that have different data types.

Variable declaration

You can declare a variable in two ways:

One is with the keyword var. This syntax can be use to declare both local and global variables, depending on the execution context.

                        
                            var my_variable = 'Boo';

Second is with the keyword const or let. This syntax can be use to declare a block-scope local variable.

                        
                            // to declare a variable:
                            let myVariable;

                            // to declare a constant
                            const CONSTANTNAME = constant_value;

Example of a variables and constants:

                        
                            // to declare a variable:
                            let Age = 25;

                            //we can also specity multiple variables with the let keyword.
                            let firstname, lastname = "David Malan";    

                            //we can still declare multile variables plus some other variable such as:
                            let firstname, lastname = "David Malan", Age = 25;

                            //to declare a constant
                            const PIE = 3.14, APPNAME = "Slack";
                            console.log(firstname);
                            console.log(lastname);
                            console.log(Age);
                            console.log(PIE);
                            console.log(APPNAME);

Literals

a literal is the value given to the identifier and the identifier is the name given to the variable.

So a literal can be a number, float, string or even the name of another variable.

In the examples above, 3.14 is the literal for PIE.

Variable Scope

When you declare a variable outside of any function, it called a global variable, because it is available to any other code in the current document.

When you declare a variable within a function, it is called a local variable, because it is available only withing that function.

Identifiers

Names of variables and constants are called identifiers.

Naming rules of identifiers

You can name your variable anything but there's some restrictions.

  • An identifier must start with a letter, underscore (_) or a dollar sign ($). Subsequent characters can also be digits (0 - 9).
  • Unicode characters are allowed.
  • An identifier cannot be a reserved word.

some examples of legal names include: Number_hits, temp99, $credit and _name.

Data Types

There's two data types in JS:

  • Primitive types - are immutable meaning you can't change the value of the variable after you initialize it.
  • Object types - there's five types of object types: array, date, regular expression, map and weakmap, set and weakset.

String - This is a sequence of text known as a string.

To signify that the value is a string, enclose it in single or double quote marks.

                        
                            let my_variable = 'Bob';

Number - This is a number. Numbers don't have quotes around them.

                        
                            let my_variable = 10;

Boolean - This is a True / False value.

The words true and false are special keywords that don't need quote marks.

                        
                            let my_variable = true;

Array

An array holds many values in a single variable.

This is a structure that allows you to store multiple values in a single reference.

It's mostly used when you want to loop through a list of items and want to do the same thing to each one of them.

                        
                            // shortest way to create an array:
                            let my_variable = [1, 'Bob', 'Steve', 10];
                            // refer to each member of the array like this:
                            my_variable[0];
                            my_variable[1];

                            // another way to create an arary:
                            let my_variable = new Array(1, 'Bob', 'Steve', 10);

Object - This can be anything. Everything in JS is an object and can be stored in a variable.

                        
                            let my_variable = document.querySelector('h1');

Symbols represents tokens.

Operators

An operator is a mathematical symbol that produces a result based on two values (or variables).

Addition (+) - Add two numbers together or combine (concatenate) two strings.

                        
                            // to add two numbers:
                            6 + 9;

                            // to concatenate two strings:
                            var words = "This is";
                            var morewords = " a sentence";
                            var sentence = words + morewords;
                            document.write(sentence);

Subtraction (-)

                        
                            9 - 6;

Multiplication (*)

                        
                            9 * 6;

Division (/)

                        
                            9 / 6;

Assignment (=) - Assigns a value to a variable.

                        
                            9 - 6;

Equality (===) - Performs a test to see if two values are equal. It returns a true / false (Boolean) result.

                        
                            let my_variable = 3;

Conditionals

These are code structures used to test if an expression returns true or not.

if ... else

                    
                        let ice_cream = 'chocolate';
                        if(ice_cream === 'chocolate') {
                            alert('Yay, I love chocolate ice cream');
                        } else {
                            alert('Awww, but chocolate is my favorite');
                        }

The expression inside the if(...) is the test. If this comparison returns true, the first block of code runs. If the comparison is not true, the second block of code after the else statement runs instead.

Functions

Functions are useful when you have code you want to re-use.

Functions are a way of packaging functionality that you wish to reuse.

It's possible to define a body of code as a function that executes when you call the function name in your code.

This is a good alternative to repeatedly writing the same code.

Some examples of functions built into the browser include:

                        
                            let my_variable = document.querySelector('h1');
                            alert('hello!');

Arguments

Functions often take arguments (bits of data needed to do the job).

Arguments go inside the parentheses, separated by commas if there is more than one argument.

Creating a function

You can define your own functions.

There's three different ways to define a function.

using the keyword function.

                        
                            // function that takes two arguments and multiplies them.
                            function multiply(num1, num2){
                                let result = num1 * num2;
                                return result;
                            }

creating a function as a variable

                        
                            const multipy = function multiply(num1, num2) {
                                let result = num1 * num2;
                                return result;
                            }

The return statement

The return statement tells the browser to return the result variable out of the function so it is available to use.

This is necessary because variables defined inside functions are only available inside those functions. This is called variable scoping.

Loops

For loop

The most used loop in JS is the for loop.

For loop executes a block of code multiple times.

Creating a for loop:

                        
                            for(i = 0; i < 5; i++){
                                document.write("This is iteration " + i + "<br>");
                            }

The i = 0 statement is a counter variable.

The i < 5 is the condition.

The i++ is the action taken after each iteration and is executed last after each iteration happens.

Events

Real interactivity on a website requires event handlers.

Event handlers are code structures that listen for activity in the browser, and run coe in response.

The most obvious example is handling the click event, which is fired by the browser when you click on something with your mouse.

Example:

                        
                            let my_html = document.querySelector('html');
                            my_html.addEventListener('click', function() {
                                alert('Ouch! Stop poking me!')
                            });

Here, we select the <html> element. We then call its addEventListener() function, passing in the name of the event to listen to ('click') and a function to run when the event happens.

Functions passed to addEventListener() are called anonymous functions, because they don't have a name.

An alternative way of writing anonymous functions, is by calling an arrow function.

An arrow function uses () => instead of function ():

                        
                            let my_html = document.querySelector('html');
                            my_html.addEventListener('click', () => {
                                alert('Ouch! Stop poking me!')
                            });
