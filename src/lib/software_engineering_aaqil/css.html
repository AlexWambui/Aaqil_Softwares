Introduction

CSS stands for Cascading Style Sheets

CSS describes how HTML elements are to be displayed on screen, paper, or in other media

CSS (Cascading Style Sheets) were invented by Bert Bos and HÃ¥kon Wium Lie, because HTML was getting less and less semantic, and more and more about presentation, which would make it less future-proof and more verbose.

CSS enforces not only semantics in the HTML, but also mutualization of the presentation layer, and therefore less code.

Here, we'll learn:

Syntax

A CSS rule-set consists of a selector and a declaration block:

The selector points to the HTML element you want to style.

The declaration block contains one or more declarations separated by semicolons.

Each declaration includes a CSS property name and a value, separated by a colon.

A CSS declaration always ends with a semicolon, and declaration blocks are surrounded by curly braces.

Example:

                        
                            p{
                                color: red;
                                text-align: center;
                            }

Comments

Comments are used to explain the code, and may help when you edit the source code at a later date.

For CSS comments, we use: /* comment */

Example:

                        
                            /* This is a single line comment */
                            p{
                                color: red;
                                text-align: center;
                            }
                            /*
                             * this is a multi line comment
                             * you can place as many as you need
                            */

Classes

Classes are reusable styles that can be added to html elements.

They allow you to use the same CSS styles on multiple html elements.

Example:

                        
                            .title{
                                color: blue;
                                font-weight: bold;
                            }

This means any elements with this class will have the styles in the class applied to it.

Variables

to create variables we use the code --variable-name: value;

to use the variable we use the code var(--variable-name, alternative-value). The alternative value called fallback value is used just in case the variable does not work.

Example:

                        
                            .penguin{
                                --penguin-skin: gray;
                                background-color: var(--penguin-skin, gray);
                            }

This sort of variable would only work within the elements specified, we can create variables that can be used allover the stylesheet by using :root{variables} to create a global variable.

                        
                            :root{
                                --penguin-bely: pink;
                                --penguin-beak: orange;
                            }

Borders

Instead of using only one one value for border-radius we can use four or even pair of four values:

                        
                            /* All corners */
                            border-radius: 32px;

                            /* top-left and bottom-right 32px; bottom-left and top-right: 64px; */
                            border-radius: 32px 64px;

                            /* for all corners */
                            border-radius: 64px 96px 128px 32px; 

                            /* first level radius, radius for eliptic corners */
                            border-radius: 40% 20% / 10% 30%;

                            /* first level radius / every corner defined */
                            border-radius: 32px 15% / 16px 32px 48px 64px;

Tables

To customize a table, we would the codes:

                        
                            table, th, td{
                                border-collapse:collapse;
                            }
                            table{
                                width: 60%;
                                margin:auto;
                                border:1px solid gray;
                            }
                            th{
                                color: black;
                                background: aquamarine;
                                padding-top: 5px;
                                padding-bottom: 5px;
                                padding-left: 5px;
                                padding: 5px auto 5px 5px; /* T-R-B-L */
                                padding: 5px 5px; /* TB - LR */
                                text-align: left;
                            }
                            td{
                                padding-left: 5px;     
                                border-bottom: 1px solid gray;  
                            }
                            tr:hover td{
                                background:gainsboro;
                            }

Grid

fr

Grid introduces a unit called fractional (fr).

Each fr unit allocates one share of the available space.

For example, if two elements are set to 1fr and 3fr repsectively, the space is divided into 4 equal shares; the first element occupies 1/4 and the second element 3/4 of any leftover space.

To learn grid, use: gridgarden.com
