Language properties of Python

Python is an interpreted language

when you run python program, an interpreter will parse python program line by line basis, as compared to compiled languages like C or C++, where the compiler first compiles the program and then start running.

So what's the difference?

The difference is that interpreted languages are a little bit slow as compared to compiled languages.

writing codes in compiled languages is a daunting task for a beginner.

Also in such languages, you need to write even most basic functions like calculate the length of the array, split the string etc.

For more advanced tasks sometimes you need to create your own data structures to encapsulate data in the program. So in C/C++ before you actually start solving your business problem you need to take care of all minor details. This is where Python comes. In Python, you don't need to define any data structure, no need to define small utility functions because Python has everything to get you started.

Python is dynamically typed

python doesn't require you to define variable data types ahead of time.

python automatically infers the data type of the variable based on the type of value it contains.

Python is strongly typed

PHP or Javascript, convert data of one type to another automatically.

For example: In JS, 1 + "2" - converts 1 to a string and concatenates to "2", which results to '12' which is a string.

However, in python, such automatic conversions are not allowed.

so 1 + "2" would produce an error.

Running Python Programs

Python uses a command-line tool called pip.

to run a program stored in a file:

  1. Open the terminal and change current working dir to dir where the file is stored
  2. to run the program, type the command:

    python3 hello.py

alternatively you can make python programs runnable scripts by using these line at the beginning of python scripts

#!/usr/bin/python3

then making the scripts executable using

chmod u+x file_name

Comments

Comments are notes which describe the purpose of the program or how the program works.

Comments are not programming statements that Python interpreter executes while running the program.

Comments can also be used to write program documentation.

In python, any line that begins with a pound sign (#) is considered a comment.

Comments at the end of a statement are called end-line comments.

Master Python

Understand why you're learning python.

Figure out how you learn best - is it through books, video tutorials or is it both methods.

So the methods to learn python include:

  • Learn the fundamentals - this includes: variables, operators, control flow (conditional statements), functions, data types, loops.
  • Once done with the fundamentals, start to learn Object Oriented Programming with Python. This includes: how to; create a class, create an instance variable, use methods, inheritance.
  • Data Structures - help build problem solving skills especially for projects. These include: stack, linked lists, trees. Also used in coding interviews to see how you solve problems.
  • Learn to solve problems - figure out how to break it down into small parts. Problem solving helps you build a way of thinking.
  • Join a community - find people on the same journey as you and connect with them. Give each other pointers to grow. You can join the pythonistacafe community.
  • Share you knowledge with others as you're learning - this helps you understand concepts you're trying to teach. You can start a Youtube channel or a blog. Explain to other people to help you know what you understand and what you don't.
  • Contribute to open source projects as you get better at python - projects include: tensorflow, flask, django, keras, youtube-dl, face-recognition, Ansible, httpie, pandas, scikit-learn.
  • Learn git and improve you git work flow.
  • Learn python frameworks like flask or django.
