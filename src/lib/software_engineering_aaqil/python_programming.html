Introduction

Python is an easy to learn, powerful programming language developed by Guido Van Rossum of the Netherlands in the 1990s.

Python was inteded to be simple, intuitive language that is as powerful as traditional languages.

Python's simple syntax and dynamic typing, together with its interpretted nature, make it an ideal language for scripting and rapid application development in many areas on most platforms.

Guido named the language Python because he was a big fan of "Monty Python's Flying Circus".

Python lets you develop programs in less time and with less code than other programming languages.

Python 2 became available in the year 2000 and it's not forward compatible (means it can't run most programs written in python 3).

Python 3 was introduced in 2008 and it's not backward compatible (means it can't run most programs written in python 2).

Why python is a great language

  • Python has a simple syntax that's easier to read and use than most other languages.
  • Python has most of the features of traditional programming langugages. As a result, you can use Python to learn the concepts and skills that apply to those languages too.
  • Python supports the development of a wide range of programs, including games, web applications and system administration.
  • Python is used by many successful companies including: Google, IBM, Disney and EA GAmes.
  • Pyton is open source, meaning its source code is available to the entire Python Community.

Types of Python Applications

  1. Console Application - is a desktop application that uses the console to interact with the user. In this type, you enter commands at the command prompt in the console available from your operating sytem. A console applicaiton being run on windows system uses the command prompt and on a Mac OS X system a console application runs in the Terminal Window.
  2. GUI Applicaiton - a GUI application performs the same tasks as the console application. It gets the same input from from the user, performs the same calculations, and displays the same results. However, GUI applications are more user-friendly and intuitive.
  3. Web Applicaiton - can be called by a web browser running on a computer or mobile device and can use a server to process or store data. A web application performs the same task as the console and GUI applications. Python frameworks such as Flask and Django can help with development of web applications.

You can also use python as a scripting language to work with other software applications or to develop system administration programs that perform tasks like automatically backing up a server or rotating log files. Programs like this usually don't have a user interface since they run in the background.

Software needed

To develop python programs, you just need to download and install Python.

It includes an IDE (Integrated Development Environment) called IDLE that you can use for coding, testing and debugging your programs.

Coding Statement

Unlike many programming lanauges, the indentation of each line matters in a Python program. Incorrect indentation causes an error.

In Python, the indentation is typically four spaces.

In some cases, you'll want to divide a long statement over two or more lines. Then, you can use implicit continuation. To do that, you divide a statement before or after an operator like a plus or minus sign.

Example of implicit continutaion:

                        
                            print("Total Score: " + str(score_total)
                                + "\nAverage Score: " + str(average_score))

The other way to continue a statement is to use explicit continuation. Then, you code a backslash to show that a line is continued. In general, this is discouraged and isn't usually required.

Example of explicit continuation:

                        
                            print("Total Score: " + str(score_total) \
                                + "\nAverage Score: " + str(average_score))

The first line in a Python program is usually a shebang line.(#!/usr/bin/env Python3).

The line is ignored by Windows, but Unix-like systems, use this line to determine what interpretter to use to run the program.

First Program

                        
                            #!/usr/bin/env Python3

                            counter = 0
                            score_total = 0
                            test_score = 0

                            while test_score != 999:
                                test_score = int(input("Enter test score: "))
                                if test_score >= 0 and test_score <= 100:
                                    score_total += test_score
                                    counter += 1

                            average_score = round(score_total / counter)

                            print("Total Score: " + str(score_total))
                            print("Average Score: " + str(average_score))

Pycodestyle

Pycodestyle is a tool to check your Python code against some of the style conventions in PEP8.

The package used to be called pep8 but was renamed to pycodestyle to reduce confussion.

Installation

You can install, upgrade and unistall pycodestyle.py using the commands:

pip install pycodestyle

pip install --upgrade pycodestyle

pip uninstall pycodestyle

Example of usage:

pycodestyle --first optparse.py

To make pycodestyle show the source code for each error, and even the relevant text from PEP8:

pycodestyle --show-source --show-pep8 testsuite/E40.py

To display how often each error was found:

pycodestyle --statistics --qq Python-2.5/Lib

Comments

Comments are usually ignored by the Python interpreter and thus have no effect on the operation of the program.

Use comments to describe portions of code that are hard to understand, but don't overdo them.

To comment, both inline and block comments, start with the # sign.

Multi line comments can also be written using ''' Comment here '''

Example of comments in python:

                        
                            # This is a block comment that
                            # extends multiple lines.

                            '''
                            This is another way to write multi line comments
                            '''

                            counter = 0 
                            test_score = 0

                            counter +=1 # this is an inline comment. Add 1 to counter

Data Types and Variables

Control Statements

Control statements help control the execution of a program.

These statements include if statemensts as well as looping statements.

Before learning control statements, it's important to understand boolean expressions.

Boolean Expressions

Boolean expressions are expressions that evaluate to either True or False based on a comparison between two or more values and that value can be assigned to a variable.

When coding boolean expressions, be sure to compare numbers with numbers and strings with strings. Mixed expressions like comparing numbers with strings won't work the way you expect them to.

You can also assign True or False to variables.

                        
                            active = True
                            active = False

Boolean expressions can be used in if statements as well as in looping statements.

Boolean expressions are usually used in conjunction with relational operators.

Examples of Boolean Expressions:

                        
                            age == 5              # variable equal to numeric literal.
                            first_name == "John"  # variable equal to string literal

                            distance >= limit     # variable greater than or equal to a variable.

                            rate / 100 >= 0.1     # expression greater than or equal to a literal.

Operators

Relational Operators

The six relational operators are:

  • == (Equal to) - returns True if both operands are equal.
  • != (Not Equal to) - returns True if the left and Right operands are not equal.
  • (Greater than) - returns True if the left operand is greater than the right operand.
  • < (Less than) - returns True if the left operand is less than the right operand.
  • = (Greater than or equal to) - returns True if the left operand is greater than or equal to the right operand.
  • <= (Less than or equal to) - returns True if the left operand is less than or equal to the right operand.

Functions

A function is a reusable unit of code that perform a specific task.

It usually contains a group of statements.

Python provides many built-in functions that do common tasks like getting input data from the user and printing output data to the console.

Calling a function

The syntax for calling a function is:

                        
                            function_name([arguments])
