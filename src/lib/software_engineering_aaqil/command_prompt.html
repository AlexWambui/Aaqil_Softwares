General Commands

cls / clear - used to clear the screen

clear

Color - to change the console color e.g. to color light green

color a

title - set title for console e.g. to set the title to $amphetamane

title $amphetamane

ver - used to display the O.S version

ver

tree - displays folder structure graphically

tree

Shutdown - to shutdown:

shutdown -s

Restart - to restart:

shutdown -r

Logout - to logout:

shutdown -l

Set Shutdown Timer - to set a timer after 50 seconds

shutdown /s /t 50

Display Shutdown Message - to display a message before shutdown:

shutdown /s /t 50 "Enter your message here"

Networking Commands

ipconfig - to view TCP / IP configuration in WINDOWS like the ip address:

ipconfig /all

ifconfig - to view TCP / IP configuration in UNIX, Linux and Mac like the ip address:

ifconfig /all

ping - used to check connectivity between networking devices

ping 192.168.0.1

to run a continued ping without stopping

ping 192.168.0.1 -t

to get the ip address of a website say google.com

ping www.google.com

arp - used to view and manage the arp cache (arp utility)

arp is both a utility and also a protocol

arp -a

tracert / traceroute - tracert : used in WINDOWS to view the entire path a packet takes to get from one device to another e.g. from a computer to a server

this can be very important when you ping and fail to get a reply so you can trace where the problem is

tracert www.google.com

tracroute : used by Unix, Linux and Mac

nslook - used by WINDOWS to troubleshoot DNS name resolution issues

has both interactive and non-interactive modes

this means you can type nslookup and be taken to a different cmd prompt to work with nslookup separately or use nslookup together with some switches at the same time

to use interactive mode

nslookup

to use non-interactive mode

nslookup google.com

dig - used by Unix, Linux and Mac for the same purpose as nslookup

nslookup can also be used in this OSs but dig is considered to be more powerful

it does not have an interactive mode

route - used to manage and display the routing table

netstat - displays TCP / IP statistics and connections

to list all ports (listening and established)

netstat -a

to list active connections e.g. when connected to the internet

netstat -o

nbtstat - displays NetBIOS statistics to assist in troubleshooting name resolution issues

not used these days

Hotspot

to create the hotspot

netsh wlan set hostednetwork mode=allow ssid=hotspotname key=password1234567

to start the hostpot

netsh wlan start hostednetwork

to stop the hotspot

netsh wlan stop hostednetwork

Websites

start / open a website

to open a website say, www.google.com

start www.google.com

Processes

running processes

tasklist

to get a detailed list

tasklist /v

Directories

List all directories

to get a list of all dirs navigate to the folder first then open cmd and:

dir

to sort the directory list

dir /b /s | sort

Secured Folder

this folder cannot be deleted or moved not unless you delete using rm command

open cmd in the folder you want to create the secured folder

cd desktop

create the secured folder

md aux\

to remove the directory

rd aux\

Hide Dir

to hide a dir even when hidden items is enabled use:

attrib *h *s *r dirname

to reveal / unhide the dir:

attrib -h -s -r dirname

List Installed Programs

to get a list of all the programs you have installed (might take some time):

wmic product get name

Ouput to Clipboard

to copy the output of a command to the clip board then paste in another program say Notepad:

dir | clip

W7 Bootable USB

  1. Open cmd as admin
  2. to configure disk partition properties

    3. diskpart

  3. to det the usb drive number or drive letter

    4. list disk

  4. to select disk with USB drive e.g. select disk 1

    5. select disk 1

  5. to delete all data from the USB

    6. clean

  6. to create new primary partition on the USB

    7. create partition primary

  7. to select the created partition

    8. select partition 1

  8. to set it as active

    9. active

  9. to format file system as New Technology File System

    if server platform supports Unified Extensible Firmware Interface (UEFI), format the USB drive as FAT32 rather than NTFS.

    to format as FAT32, use fs = fat32 quick

    10. format fs = ntfs quick

  10. to exit

    11. exit

  11. copy the boot folder to the USB

Tips

Shortcuts

TAB : to autocomplete or see the dirs in the current folder

F7 : to view history of currently used commands

WIN + X : to run cmd as admin

Check if running as admin:

to check if you are running cmd as an admin:

net session

if the result is access denied then it means you're not running cmd as an admin
