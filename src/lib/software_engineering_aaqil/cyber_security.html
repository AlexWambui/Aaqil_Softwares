Introduction

Introduction to Hacking

Hacking refers to an array of activities which are done to intrude someone else's personal private information space so as to use it for malicious, unwanted purposes.

Hacking also referes to activities aimed at exploiting security flaws to obtain critical information for gaining access to secured networks.

Hackers

Hacker was originally meant a very gifted programmer who could push a program beyond what it is designed to do.

A hacker is someone who bypasses the system's access controls by taking advantage of security weaknesses left in the system by developers.

Cracker

There are 3 groups:

  1. Vandals - hack comp systems for destruction (deleting files)
  2. Jokers - the most harmless. They hack systems and carry in different sounds, noises and visual effects.
  3. Breakers - professional criminals who commit hacking of computer systems with the purpose of money theft, industrial or commercial espionage and theft of expensive software or destroy web pages.

Roadmap

How to become a hacker

  • Get familiar with Kali Linux - comes equiped with over 600 tools for hacking, cracking passwords and penetration. It's open source so you can tweak it to your preferences, thus called hacker's paradise.
  • Learn Computer Networking - understand what goes on in the background such as the protocols so you are able to manipulate them. It would be difficult to gain access to networks if you don't understand how they work.
  • Learn Computer Programming - learn python as it is very essential for hacking.
  • Learn how to scan and gather valuable information - A hacker without information is useless. You should be able to gather information on networks, websites, systems so as to know how to get in and out.
  • Learn about various server side attacks - these include sql injections, DOS, etc. This helps you learn how to initiate and stop the attacks when necessary.
  • Learn the popular hacking tools - these include: metasploit, John the ripper, BeEf, BurpSuite, Nmap, Wireshark, Sqlmap.
