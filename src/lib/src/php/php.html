Servers (localhosts)

software that provide a server and a database locally include:

  • WAMP
  • MAMP
  • LAMP
  • XAMPP

After installing xampp you can open the xampp folder in the c drive for Windows and go to xampp_control.exe so you can create a shortcut for the xampp dashboard.

Configuring Apache

this avoids conflicting with other applications

  • open the dashboard
  • click on config button for Apache
  • click on Apache (httpd.config) option
  • find listen in the file opened
  • change #listen 12.34.56.78:80 to something like #listen 12.34.56.78:9090
  • change the listen below to 9090 (same as the listen above)
  • find servername localhost
  • use the same port number as that for listen
  • stop apache then start it again on the dashboard
  • on the browser, go to the phpmyadmin page and ensure the new port is what is shown on the address bar.

after setting up the listen port for apache, open the config for xampp

  • click on service and port settings
  • for apache, set the port number to the one you set in the httpd.config file for apache.
  • click on save
  • click on admin for apache to ensure the xampp admin page shows up without errors.

Introducing PHP

PHP is an acronym for "PHP: Hypertext Preprocessor"

PHP files have extension ".php"

PHP scripts are executed on the server.

A PHP file can contain text, HTML, CSS, JavaScript and PHP code.

PHP runs on various platforms:

  • Windows
  • Linux
  • Mac OS x

Servers compatible with PHP are:

  • Apache
  • IIS

Syntax

PHP code has to be placed between two tags:

<?php - is the opening tag

?> - is the closing tag

Comments

Single line comments use: //single line comment

multiple line comments use: /*multi line comments */

echo

is used to display:

  • text
  • numbers
  • HTML
  • Data from Database
  • Data from forms

example:

                            echo "This is a php text";
                            echo 45;
                            echo "<p>This is a html paragraph</p>";

                            45
                            This is a html paragraph

Variables

A variable is a word you choose that will store values(text-string or number-integer) in it.

Declaration & initialization

to declare we use $ before the variable name.(e.g)$name

to initialize we use the assignment operator.(e.g)$name = "Chris Aaqil"; or $age = 45;

Strings are placed inside " " or ' '. This mean "25" would be interpreted as a string instead of an integer.

Integers don't have any " " or ' '. Otherwise they would be strings.

Logical Operators

these include: && And, || Or, ! Not. (e.g)is $x and $y true?

are used to evaluate two statements.

Example: is password and username entered into a form same as password and username stored in the database?

Comparison Operators

these include: == Equal, != Not equal, > greater than, >= greater than or equal to, < less than, <= less than or equal to.

are used to check for equality, greater or less than.

Example

                            $name = "Chris Aaqil"
                            $password = "Ami787sk98sk443w";
                            $minpassword = 8;

                            if (strlen($password) < $minpassword){
                                echo "password is less than minimum requirement";
                            }
                            else{
                                echo "name is ".$name." and password is ".$password." ";
                            }
                        
                            name is Chris Aaqil and password is Ami787sk98sk443w

Control Statements

are used when we want to execute code based on a given condition.

to do this we use if, if...else..., if...else if...else

Arrays

are used to assign multiple values (both integers and strings) to a single variable

Example:

$users = array("Paul", "United States", 21, "Dubai");

we use keys (index value of an element in an array) to access elements in the array.

To access "Paul" we would use:

$users[0];

Types of arrays:

Two types of arrays in php are:

  • indexed arrays -
  • associative arrays - when you want to assign the keys with your value / key. so you don't want php to used indexed keys / values to reference your values.
  • multidimensional

associative arrays

example of an associative array

$users = array("fullname" => "Paul", "nationality"=>"Cuban", "age"=>21);

in this example, "fullname" becomes the key for "Paul".

to print the fullname we would use:

echo $users['fullname'];

Loops - foreach

in verb form, a loop is something that circles.

loops are mostly used to display data say from databases as it enables looping through all records.

Functions

used to hold a block of statement we want to use repeatedly instead of having to repeat the same code over and over again.

functions by default don't execute automatically so we have to call them.

naming a function

to name functions we either use lowercase to separate words, or use camel case.

Example:

function authenticate_user() or function authenticateUser()

calling a function

assuming we have a function such as:

                            function authenticate_user($password, $formpassword){
                                if ($password == $formpassword){
                                    header('location: myaccount.php");
                                }
                            }

we would call the function using:

authenticate_user();

Examples of PHP inbuilt functions:

  • foreach()
  • mysqli_fetch_assoc()
  • mysqli_num_rows()

Arguments

if the function uses variables, we have to pass the variables into the function as arguments.

when calling a function with arguments, we have to pass the arguments to the function otherwise we would have errors.

Include

include is a functions that enables you to include other scripts into the current page.

it parameter is the path of the file to include.

example:

include ("functions.php");

Require

works the same as include but the difference is if require is used then the program crashes if the file required is not found.

Super Globals

they include:

  • $GLOBAL - makes a declared variable accessible to all your php files. Globals are usually arrays. (e.g)$GLOBAL['user'] = "Paul Ambrose"
  • $_SESSION - keeps all variables throughout the duration of your application until destroyed. $_SESSION variables are usually arrays.
  • $_POST - used to collect data from forms
  • $_GET - used to collect data from forms through the url
  • $_FILES - used to collect files from forms
  • $_SERVER - keeps paths and headers

using $_SESSION super global

since $_SESSION is an array, we can declare it as:

                            $_SESSION['user_info'] = array(
                                "full_name" => "Paul Ambrose",
                                "email" => "paul@gmail.com"
                            );

to echo or use the session super global we would use:

$_SESSION['user_info']['full_name'];

Filters & Sanitization

validation is when we check if proper data is entered

sanitization is when we remove illegal characters

inbuilt validation functions:

  • htmlspecialchars();
  • stripslashes();
  • trim();
  • filter_var(variable, filtername, option); - does both validation and sanitization.

sanitization

using filter_var(variable, filtername, option);

example:

                            $raw_email = trim($_GET['email']);
                            $clean_email = filter_var($raw_email, FILTER_VALIDATE_EMAIL);
                            /*so in the code we should now use $clean email*/

Concatenation

means adding (or joining) strings together.

in php we use a period(.) to concatenate when we want to include our variables in a string.

example:

echo "Your email is ".$clean_email

to put the email in the same string:

echo "Your email is ".$clean_email." you can now login";
