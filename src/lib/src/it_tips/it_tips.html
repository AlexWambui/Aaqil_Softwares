Startup Programs

To add programs to the startup menu:

  • Win + R
  • shell:startup
  • add a shortcut for the program you want to run on startup

To view which program are starting up when the computer boots

  • Win button
  • type startup
  • a list of all startup programs will show including their impact (high or low impact)

Shortcuts

  • ctrl + a = select all
  • ctrl + b = bold
  • ctrl + c = copy
  • ctrl + d = duplicate
  • ctrl + e = center align
  • ctrl + f = find
  • ctrl + g = group & go to
  • ctrl + h = replace
  • ctrl + i = italic
  • ctrl + j = justify
  • ctrl + k = hyperlinks
  • ctrl + l = left align
  • ctrl + m = new slide
  • ctrl + n = new document & page
  • ctrl + o = open
  • ctrl + p = print
  • ctrl + q = quit
  • ctrl + r = right align
  • ctrl + s = save
  • ctrl + t = new tab
  • ctrl + u = underline
  • ctrl + v = paste
  • ctrl + w = close the window
  • ctrl + x = cut
  • ctrl + y = redo
  • ctrl + z = undo
  • ctrl + Q - quit your web browser.
  • Alt + left click an image to automatically download.
  • Windows + C - activate Cortana.

Colors

White has 100% color brightness, and black has 0% such a disparity creates intense light levels that overstimulate the eyes.

pure black text on white backgrounds can cause eye strain when users read the text over an extended period.

survey found 58% of adults in the U.S have experienced eye strain from working on computers
