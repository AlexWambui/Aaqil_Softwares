Introduction

Web Development is usually divided into two categories:

  • Front-end
  • Back-end

To understand the difference between the two, consider a web page. There's what you see on the web page and there's where the data comes from to make it possible for the content on the page to be dynamic.

So in simpler terms, web applications are just a constant communication between the broswer and the server.

Programming done to display content on the browser as seen by the users is called front-end while programming done to ensure servers can send data from the server to the browser so it provides dynamic data to the user on the browser is referred to as back-end.

Other technologies to learn include:

  • JS libraries
  • JS Frameworks
  • Command line
  • Module loading / bundling tools (Webpack, browserify, gulp to minify code so it's loaded quickly and efficiently into the browser).
  • Wordpress
  • Image editors (photoshop)
  • Wireframe tools (plans or sketches eg. Figma)

Introduction to HTML Tags

In HTML there's over 100 tags but we only use like 10 or 15.

Headings

there's usually 6 heading tags. <h3 class="_heading3" class="_heading2"> - <h6>

Paragraphs

to create paragraphs, use the <p> tag.

most of the time a placeholder for paragraphs called lorem... is usually used, especially when you don't have all the content at the moment.

Images

to insert images, use: <img src=" " alt=" ">

the most professional way to insert images is from another website, right click and copy the image address of that image and not the link address.

image from another website

                                                <img src="https://images.pexels.com/photos/6662831/pexels-photo-6662831.jpeg?auto=compress&cs=tinysrgb&dpr=1&w=500" 
                                                alt="imaginary image using link address" height="200">

to include images from a folder on the same website:

image from same website

                                        <img src="../images/laptopcode.jpg" alt="laptop code" height="200">

images can be styled inline by either using height of width attribute alone (to let images scale themselves).

you can scale the image upwards but take care of images not being too pixelated.

alt in images can be used to describe what's happening in the images esp for the visually impaired since it's read out to the person.

some places to find images include:

  • icon finder
  • pixels

Sup & Sub

to include a superscript use the <sup> and to include a subscript use <sub>.

Bold & italics

to make text be bold: <b>.

to make text be italic: <i>

Horizontal rule

to draw a horizontal rule, use the <hr/>

Break tag

the break tag: <br/> is used to enable elements to start on a new line.

Attributes

usually give more information about the tags

the order of the attributes usually doesn't matter.

Notes

too many web pages take up memory

html appears according to the code in the html

laptops have pixels which give the resolution

CSS stands for Cascading Style Sheets.

this is because styling is usually done starting from the outermost element and cascading to the innermost element.

Styling mainly consists of:

                                                
                                                    property: value;

using inline CSS

created a paragraph then used inline css to change the color, font-family and font-size.

using internal CSS

this is done using the <style> tag usually placed inside the <head> tag.

Commenting in CSS

usually done to tell other developers what you're doing esp when the code becomes a little bit complex.

can also include commenting out / disabling lines of code.

Beginner friendly projects

1. A Tribute page

The simplest website you can make as a beginner is a tribute page of someone you admire in your life.

Make a webpage writing about that person adding his/her image. On the top of the webpage, add the image and name of the person and below that give layout for the rest of the details. You can use paragraphs, lists, links, images with CSS to give it a descent look.

2. Webpage Including Form

Forms are always an essential part of any project and you will be working with a lot of forms in most of the applications.

How to use a text field, checkbox, radio button, date, and other important elements in a single form.

3. Parallax Website

A parallax website includes fixed images in the background that you can keep in place and you can scroll down the page to see different parts of the image.

Using the parallax effect in web designing is really popular and it gives beautiful look and feels to the webpage.

divide the whole page into three to four different sections. Set 3-4 background images, align the text for different sections, set margin and padding, add background-position and other CSS elements and properties to create a parallax effect.

4. Landing Page

You will practice how to add footer and header, create columns, align-items, divide the sections and a lot of things.

Color combinations should go well with each other for different sections or backgrounds.

5. Restaurant Website

Showcase your solid knowledge of HTML and CSS creating a beautiful webpage for a restaurant.

You will be aligning the different food items and drinks using a CSS layout grid. You will be adding prices, images and you need to give it a beautiful look and feel as well using the proper combination of colors, font-style and images.

You can add pictures gallery for different food items, you can also add sliding images for a better look.

6. An Event or Conference Webpage

You can make a static page holding an event or conference.

People who are interested in attending the conference create a register button for them. Mention different links for speaker, venue and schedule at the top in the header section. Describe the purpose of the conference or the category of people who can get benefit from this conference. Add an introduction and images of the speaker, venue detail and the main purpose of the conference on your webpage.

7. Music Store Page

Add a suitable background image describing the purpose or what the page is all about. In the header section add different menus. Add buttons, links, images and some description about the collection of songs available.

At the bottom mention the links for shopping, store, career or contact details.

8. Photography Site

Make a one-page responsive photography site.

Add the company name with an image (related to photography) on the top (landing page).

Mention the contact detail of the photographer at the bottom (footer). Add a button to view your work. This button will directly bring you down to the images section.

9. Personal Portfolio

Showcase your work samples and skills in your portfolio with your name and pictures. You can also add your CV there and host your complete portfolio on GitHub account.

At the top add one of your images and introduce yourself there. Below that add some work samples and at last (footer) add contact information or social media account.

10. Technical Documentation

Divide the whole webpage into two sections. The left side creates a menu with all the topics listed from top to bottom. Right side you need to mention the documentation or description for the topics. The idea is once you click on one of the topics in the left section it should load the content on the right.

Job Application

Once you've mastered HTML, CSS and JS, start applying for jobs even if you don't meet all the listed requirements.

Most of the time even if you fail to get the job, you'll at least get the experience of going through the interview process and get feedback on what you need to work on.

If you don't know something, be honest and show that you do not know but show that you know how to get the solution.
