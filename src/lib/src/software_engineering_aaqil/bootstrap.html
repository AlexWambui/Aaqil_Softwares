BS Grid

The BS Grid allows you to easily control how your website layout will be displayed on different types of devices.

The grid system is a series of up to 12 columns that scale, contract and reposition themselves based on the size of the device screen or viewport.

Column sizes:

  • xs - extra small. Usually for phones.
  • sm - small. Usually for tablets.
  • md - medium. Usually for laptops / desktops.
  • lg - larger. Usually for larger desktops.

To make the grid responsive:

col-sm-6 col-md-4 col-lg-8
